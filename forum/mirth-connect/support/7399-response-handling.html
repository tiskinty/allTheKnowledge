    You can do something like this in the postprocessor. Pull the Response object from the response map, get the message with getMessage(), and then use the built-in HL7->XML conversion template to create an E4X XML object out of it.

    However, it's important to note that the postprocessor runs after the destination has processed the message. An HTTP Sender doesn't care what's in the payload coming back, it just looks to see what response code is returned. So if an HL7 NACK is wrapped in a 200 HTTP response, then the destination connector message will still be marked as SENT rather than ERROR. You can get around this though with some code in the postprocessor that manually updates the message status and channel statistics. (Search the forums; there are at least a few examples)

                            Thanks for your input. I've included the function that is included in a code template that is called from my destination. The HTTPSSender is a java class that we wrote to accomplish the SSL piece because we are not using the appliance. I can post more, but I'll have to untangle some stuff. Thanks again in advance.

                            // modify function_name and parameters as you wish.
                            // one function per template is recommended. i.e.) create a new code template for each new function.
                            function VxuSendHttpsPostClientCert(state, url, certificatePath, certificatePassword) {
                            // write code here.

                            var messageBody = messageObject.getEncodedData();
                            var httpsSender = new Packages.com.xti.util.HTTPSSender(certificatePath, certificatePassword);

                            // Set all the necessary parameters...
                            httpsSender.setURL(url);
                            httpsSender.setMessageBody(messageBody);

                            logger.info("Sending message to " + url + " - " + messageBody);

                            // Send the message to the URL...
                            try
                            {
                            httpsSender.send();
                            var retString = httpsSender.getReturnValue();
                            retString = retString.substr(0, retString.indexOf("MSA")) + "\r" + retString.substr(retString.indexOf("MSA"));
                            var msgT = new Packages.com.mirth.connect.model.converters.Serial izerFactory.getHL7Serializer().toXML(retString);
                            channelMap.put('msgT',msgT);

                            if (msgT['MSA']['MSA.1']['MSA.1.1'].toString() == 'AA') {
                            logger.info("HTTPS send success. Return value=" + retString);
                            }
                            else {
                            logger.info("HTTPS send failure. Return value=" + retString);
                            throw("HTTPS send failure. Return value=" + retString);
                            }
                            }
                            catch (e)
                            {
                            throw(e.message);
                            }
                            }
