    Hi,

    Can someone point me to good references for creating fairly complex XML output please.

        Assuming the final XML should like:

        <batch>
        <report>
        ...
        ...
        </report>
        <report>
        ...
        ...
        </report>
        </batch>

        structure, Pls try this:

        var payload = ["a","b","c","d"]; // I assume this is the rs
        var batch = new XML(<batch/>);
        var report = new XML(<report/>);
        var reportIdx = payload.length;

        for ( i = 0; i < reportIdx; i++){
        report.@icn = i;
        report.data = payload[i];
        batch.appendChild(report);
        }

        It gives the following:

        <batch>
        <report icn="0">
        <data>a</data>
        </report>
        <report icn="1">
        <data>b</data>
        </report>
        <report icn="2">
        <data>c</data>
        </report>
        <report icn="3">
        <data>d</data>
        </report>
        </batch>

        This has to be a javascript transformer, and nothing to be defined under MessageTemplate. Please keep the Outbound MessageTemplate as XML

        HTH

        Thomas
