    You can loop just the OBXs.

    Code:
    var Dx = ''
for each (obx in msg.OBX)
  if (obx['OBX.3']['OBX.3.1'].toString() == 'GI FINAL DX' && obx['OBX.5']['OBX.5.1'].toString().length > 0)
    Dx += obx['OBX.5']['OBX.5.1'].toString() + '\n'
channelMap.put('Dx',Dx)

      Your loop logic is traversing every segment in the message and checking the OBX fields on every segment encounter (MSH, PID, PV1...etc). I think you only want to check them when you encounter OBX segments. To keep your loop logic the same, you might check for the segment name first:

      Code:
      if (seg.name() == 'OBX')
      then check seg['OBX.3'] and seg['OBX.5']
