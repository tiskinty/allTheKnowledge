      Re: Unable of start Mirth 1.3

      I did a quick test (File reader to database) and Mirth v1.3 worked fine (Windows 2000 SP4 w/ current updates, Java Runtime Version 1.5.0 Update 9 [ build 1.5.0_09-b03 ] ).

      Here&#039;s what I did:

      1. Cleared Temporary Internet Files (Control Panel / Java / General tab / Delete Files)
      2. Cleared Temporary Internet Files from Internet Explorer
      3. Downloaded and extracted Mirth-v1.3 zip file
      4. Started Mirth.bat
      5. Launched http:localhost:8080 from Internet Exporer
      Got prompted to save webstart / Saved to a local folder / Accepted prompt to trust mirth.org (don&#039;t remember exact name) /

      6. Logged in Mirth admin using admin/admin
      7. Imported channel from v1.1 / Accepted prompt to convert the channel to v1.3
      8. Modified Directory-to-read and Move-to-directory
      9. Copied 10 HL7 message files in INBOUND folder (S12, S14, A31)
      10. All 10 files were processed successfully
      11. Patient and Appointment table in the database had correct data.
