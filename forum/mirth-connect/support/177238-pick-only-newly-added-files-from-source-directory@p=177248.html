                You would use that code in a transformer (or filter) of type javascript.

                If a filter, you can just use simple if then logic. Your objective in the filter is to return a true or false boolean value:

                PHP Code:

                var x = 0;
                if (                x == 1) {
                    return                 true
                } else {
                    return                 false
                } 

                The message will filter. I will leave the actual code to you for evaluating the date / number of days. Again if you return a "true" (or nothing) the message will proceed and a false makes the message filter so thereby not go to your destinations.

                    3. You are not comparing the same date formats (which inherently bullet 2 would fix)
                    4. You are mixing date object and date string in comparisons (again bullet 2 when correct would fix this problem

                    So write a piece of code and paste it here that address those bullets. Paste screenshots. In general I don't import channels for random folks on the Internet.

                    Also see https://forums.mirthproject.io/forum...-function-help

                    And here is that same function modified to do minutes:

                    PHP Code:
                    /**
                        Modify the description here. Modify the function name and parameters as needed. One function per
                        template is recommended; create a new code template for each new function.

                        @param {Any} begin - 
                        @param {Any} end - 
                        @param {Any} pattern - 
                        @return {Any} 
                    */
                    function minutesBetween(begin,end,pattern) {
                                            importPackage(org.joda.time);
                                            importPackage(org.joda.time.format);
                                            String.prototype.toDate = function(pattern) {return DateTimeFormat.forPattern(pattern || 'yyyyMMddHHmmss').parseDateTime(this);};
                        return                     Minutes.minutesBetween(convert(begin),convert(end)).getMinutes();
                        function                     convert(object) {
                            if (                    'string' === typeof object || object instanceof String)
                                return                     object.toDate(pattern);
                            else if (                    object instanceof Date || object instanceof java.util.Date)
                                return new                     DateTime(object.getTime());
                            else 
                                return new                     DateTime(object);
                        }
                    }
