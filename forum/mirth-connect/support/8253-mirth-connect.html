        On your TCP Listener, generally you want to use "127.0.0.1" for the Listener Address, not the actual IP of your box. You want to use a specific IP only when the listening socket should be bound to a difference device aliased by the NIC.

        Also, Channel Reader is used to pick up messages used by a Channel Writer from a channel on the same Mirth Connect instance. If you want to use a File Writer/Reader on both systems, then SYS1 will need to write to a directory that SYS2 has access to, and SYS2's File Reader will need to be pointed to that directory.
